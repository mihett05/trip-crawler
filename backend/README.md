# Backend

В этой папке хранится код для service и parser. Такой подход был выбран для переиспользования кода и моделей внутри контрактов.

## Запуск

### Локальная разработка
```bash
make up  # Запуск инфры в compose 
make run # Запуск приложения
```

### Разработка в контейнере
Запуск инфры и сервиса с dlv на 40000 порту и air с hot reload
```bash
make up-debug
```

## Генерация тестовых данных
Для генерации связных тестовых данных (российские города, станции, поездки на 30 дней вперёд с полной связанностью графа) используйте:

```bash
go run scripts/generate_fully_connected_test_data.go
```

Скрипт автоматически создаст схему в Dgraph и заполнит её фейковыми данными с обеспечением полной связанности графа.

## Кодген
Предварительно нужно установить зависимости кодгена
```bash
make install
```

Запуск кодгена
```bash
make gen
```
